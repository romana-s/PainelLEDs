# PainelLEDs
O seguinte código foi desenvolvido a partir de um teste. Segue abaixo informações da questão:

Uma empresa fabrica painéis de LED compostos por quadrados de um centímetro de lado. Nos vértices de cada quadrado ficam um LED, sendo que o tamanho de cada painel é escolhido pelo cliente conforme a sua necessidade. A imagem abaixo mostram painel de 2 cm por 4 cm com posso por 15 LEDs no total. 

Atualmente os funcionários desta fábrica perdem muito tempo (que poderia ser utilizado para inovação) com cálculo manual desses painéis. A alta diretoria da fábrica decidiu então contratar você para evoluir e  construir um cálculo automatizado, modernizando assim o sistema deles.

Considerando um painel de n por m centímetros, desenvolva um código para calcular numero total de LEDs no painel.
